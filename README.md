# SolonCode Studio

> ⚠️ **非官方项目** — 本项目是社区开发者开发与维护的基于 SolonCode 第三方桌面工作台，**非 SolonCode 官方产品**。项目独立开发和发布，如有问题请在本仓库提 Issue。

[SolonCode 官网](https://solon.noear.org/article/soloncode) | [SolonCode Studio 官网](https://soloncode.studio) | [下载最新版本](https://soloncode.studio) | [使用说明](https://soloncode.studio)

SolonCode Studio 是一个基于 Tauri 2 构建的轻量桌面工作台，用于管理 SolonCode CLI 的安装、更新、卸载，以及工作区的管理和 Web/CLI 界面访问。
